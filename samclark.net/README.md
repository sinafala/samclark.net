# [samclark.net](https://samclark.net)

Code for Samuel Clark's Web Site.

## Site guide & house style

How the site is structured and the conventions to follow when editing it — the
SSI page shell, the `sam-*` content classes, `site/css/sam-styles.css`, routing
and SEO, browser workarounds, and the new-page checklist — are documented in
**[`samclark.net.md`](samclark.net.md)**. Read it before adding or editing a
page so new pages stay consistent with the rest of the site.
