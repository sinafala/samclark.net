# [samclark.net](https://samclark.net)

Code for Samuel Clark's Web Site.

## House style

The site follows a single, documented house style — the SSI page shell, the
`sam-*` content classes, the shared `site/css/sam-styles.css` stylesheet, and the
routing/SEO conventions. It is written up in **[`house-style.md`](house-style.md)**.
Read that before adding or editing a page so new pages stay consistent with the
rest of the site.
